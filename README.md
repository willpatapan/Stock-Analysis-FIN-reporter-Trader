This trading bot automatically simulates trading AAPL stock with $10,000 starting capital - no user input needed. 

How it works:
Signals: Uses 5 technical indicators (RSI, MACD, Moving Averages, Bollinger Bands, Stochastic) - needs 60% agreement to trade
Position Sizing: Invests 10-25% of portfolio per trade based on signal strength (min/max logic)
Risk Management: Auto-sells at 5% loss (stop-loss) or 10% gain (take-profit)
Performance: Tracks all trades, calculates returns, win rate, and compares vs buy-and-hold
